# Рекомендации для внедрения — ИБОиЦТ КИУ

**Назначение:** инструкции для агента/разработчика по SEO-оптимизации ibo.ieml.ru

---

## Общая схема

Сайт: https://ibo.ieml.ru
CMS/движок: (уточнить у разработчика)
Все правки вносятся через CMS или шаблоны (общие файлы: header.tpl, footer.tpl, индивидуальные страницы).

---

## Термины

- title: тег `<title>...</title>` в `<head>`
- meta description: `<meta name="description" content="...">`
- meta keywords: `<meta name="keywords" content="...">`
- h1: `<h1>...</h1>` на странице (ровно 1 шт)
- h2-h3: подзаголовки
- JSON-LD: микроразметка Schema.org в тегах `<script type="application/ld+json">...</script>`

## JSON-LD — организация (на всех страницах)

Вставить в <head> на всех страницах:

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "EducationalOrganization",
  "name": "Институт бизнес-образования и цифровых технологий",
  "alternateName": "ИБОиЦТ КИУ",
  "url": "https://ibo.ieml.ru",
  "logo": "https://ibo.ieml.ru/images/logo.png",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "ул. Островского, 67",
    "addressLocality": "Казань",
    "addressCountry": "RU"
  },
  "telephone": "+78432330023",
  "sameAs": []
}
</script>

## JSON-LD — курс (на карточках курсов)

Вставить на страницах /ru/training_calendar/?id=N:

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Course",
  "name": "[Название программы]",
  "description": "[Описание программы]",
  "provider": {
    "@type": "Organization",
    "name": "ИБОиЦТ КИУ"
  },
  "offers": {
    "@type": "Offer",
    "price": "[Цена]",
    "priceCurrency": "RUB"
  }
}
</script>

## Приоритеты внедрения

### фаза 0 (критично, 3 дня)
1. title на всех кластерных страницах (12 шт)
2. meta description на всех кластерных страницах (12 шт)
3. keywords на всех кластерных страницах (12 шт)
4. исправить h1 на странице MBA (убрать 4 h1, оставить 1)

### фаза 1 (структура, 2 недели)
5. h2 на педагогике (минимум 3 штуки)
6. h2 на IT (минимум 3 штуки)
7. alt-тексты на всех изображениях
8. breadcrumb JSON-LD на всех страницах

### фаза 2 (контент, месяц)
9. вступления на каждую кластерную страницу (200-300 слов)
10. faq-блоки с FAQPage JSON-LD
