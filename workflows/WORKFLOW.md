# Рабочий процесс: обложка + статья для ИБО

## Входные данные (всегда спрашивать):
1. тема / название программы
2. цель (привлечь заявки, объяснить, продать курс)
3. аудитория (руководители, педагоги, госслужащие, студенты)
4. формат (пост в соцсети, статья на сайт, лендинг)
5. наличие логотипа (да/нет, какой вариант)

---

## Шаг 1: SEO-подготовка
- выбрать кластер из SEO-шаблонов
- определить основной и长尾ный запрос
- собрать title, h1, description
- проверить: заголовок читаемый, не длинный, содержит ключ

## Шаг 2: Структура текста
- вступление: проблема или вопрос аудитории
- тело: 3-5 пунктов с фактами, преимуществами, процессом
- блок доверия: лицензия, опыт, эксперты
- конверсионный блок: призыв к действию, телефон, ссылка

## Шаг 3: Визуальная концепция
- выбрать композицию: карточка с иллюстрацией / инфографика / блок с цифрами
- определить цветовую схему по бренд-гайду
- решить: лого на белом фоне или на синем
- подготовить текст заголовка для обложки (крупный, читаемый)

## Шаг 4: Генерация обложки
- промпт по шаблону (см. ниже)
- проверить: читаемость текста, соответствие стилю, наличие логотипа
- если текст не читается — перегенерировать или подправить

## Шаг 5: Валидация (cathedral gate)
- [ ] стиль соответствует бренд-гайду
- [ ] seo-запросы на месте
- [ ] текст читаемый и понятный
- [ ] логотип не на зашумлённом фоне
- [ ] нет канцелярита
- [ ] есть призыв к действию
- [ ] формат подходит под площадку

## Шаг 6: Публикация / передача
- сохранить файлы в /home/workspace/Projects/ibo-growth-studio/output/
- назвать: [кластер]-[дата]-[тема].[png|md]
- сообщить пользователю: готово + что сделано

## Промпт для генерации обложки (шаблон):

```
создай обложку для статьи института бизнес-образования и цифровых технологий

стиль:
- чистый светлый фон
- синяя образовательная палитра
- мягкие тени
- много воздуха
- одна главная карточка
- плоская векторная графика
- крупный русский заголовок
- круглый логотип с совой в правом верхнем углу
- официальный академический вид

текст на обложке: [ЗАГОЛОВОК]
подтекст: [ПОДЗАГОЛОВОК]

формат: 1:1
```

## Промпт для статьи (шаблон):

```
напиши статью для сайта института бизнес-образования и цифровых технологий (ibo.ieml.ru)

тема: [ТЕМА]
кластер: [КЛАСТЕР]
аудитория: [АУДИТОРИЯ]
цель: [ЦЕЛЬ]

требования:
- заголовок h1 содержит основной seo-запрос
- вступление отвечает на вопрос аудитории
- 3-5 разделов с подзаголовками h2
- блок доверия: ссылка на лицензию, опыт, экспертов
- конверсионный блок: призыв + телефон + ссылка
- тон: уверенный, академичный, но живой
- без канцелярита, без воды
- длина: 800-1500 слов
```
